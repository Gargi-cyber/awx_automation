# AWX Network Automation (Self-Learning)

This repository contains Ansible playbooks and inventory files created for self-learning and testing AWX automation workflows.  
It demonstrates:
- How to manage network devices using Ansible.
- How to integrate GitHub with AWX for automation.
- Basic playbooks for tasks like VLAN configuration and connectivity checks.

## Features
- Example playbooks for network automation.
- Inventory structure for switches and routers.
- AWX integration steps.
